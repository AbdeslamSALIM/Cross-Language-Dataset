Règlement (CE) n o  1506/2006 de la Commission du 11 octobre 2006 modifiant le règlement (CE) n o  32/2000 du Conseil afin de tenir compte des modifications apportées au règlement (CEE) n o  2658/87 du Conseil relatif à la nomenclature tarifaire et statistique et au tarif douanier commun 
Règlement (CE) no 1506/2006 de la Commission 
du 11 octobre 2006 
modifiant le règlement (CE) no 32/2000 du Conseil afin de tenir compte des modifications apportées au règlement (CEE) no 2658/87 du Conseil relatif à la nomenclature tarifaire et statistique et au tarif douanier commun 
LA COMMISSION DES COMMUNAUTÉS EUROPÉENNES, 
vu le traité instituant la Communauté européenne, 
vu le règlement (CE) no 32/2000 du Conseil du 17 décembre 1999 portant ouverture et mode de gestion de contingents tarifaires communautaires consolidés au GATT et de certains autres contingents tarifaires communautaires, définissant les modalités d'amendement ou d'adaptation desdits contingents et abrogeant le règlement (CE) no 1808/95 [1], et notamment son article 9, paragraphe 1, point a), 
considérant ce qui suit: 
(1) Dans la nomenclature combinée de 2006, établie par le règlement (CEE) no 2658/87 du Conseil du 23 juillet 1987 relatif à la nomenclature tarifaire et statistique et au tarif douanier commun [2], modifié par le règlement (CE) no 1719/2005 de la Commission [3], les codes (codes NC) ont été modifiés pour certains produits. Les annexes IV et V du règlement (CE) no 32/2000 se réfèrent à certains de ces codes NC. Il convient dès lors d'adapter ces annexes. 
(2) Il y a donc lieu de modifier le règlement (CE) no 32/2000 en conséquence. 
(3) Étant donné que le règlement (CE) no 1719/2005 est entré en vigueur le 1er janvier 2006, il convient que le présent règlement s'applique à compter de la même date. 
(4) Les mesures prévues par le présent règlement sont conformes à l'avis du comité du code des douanes, 
A ARRÊTÉ LE PRÉSENT RÈGLEMENT: 
Article premier 
Les annexes IV et V du règlement (CE) no 32/2000 sont modifiées conformément à l'annexe du présent règlement. 
Article 2 
Le présent règlement entre en vigueur le jour suivant celui de sa publication au Journal officiel de l’Union européenne. 
Il s’applique à compter du 1er janvier 2006. 
Le présent règlement est obligatoire dans tous ses éléments et directement applicable dans tout État membre. 
Fait à Bruxelles, le 11 octobre 2006. 
Par la Commission 
László Kovács 
Membre de la Commission

ANNEXE 
Les annexes IV et V du règlement (CE) no 32/2000 sont modifiées comme suit: 
1) Dans la première partie de l'annexe IV, les codes NC de la deuxième colonne sont modifiés comme suit: 
a) en ce qui concerne le numéro d'ordre 09.0104, les codes NC sont modifiés comme suit: 
i) le code NC %quot%94051099%quot% est remplacé par le code NC %quot%ex94051098%quot%; 
ii) le code NC %quot%94056099%quot% est remplacé par le code NC %quot%ex94056080%quot%; 
iii) le code NC %quot%94059990%quot% est remplacé par le code NC %quot%ex94059900%quot%; 
b) en ce qui concerne le numéro d'ordre 09.0106, le code NC %quot%62044990%quot% est remplacé par le code NC %quot%ex62044900%quot%; 
2) Dans la deuxième partie de l'annexe IV, les codes sous le numéro d'ordre 09.0104 sont modifiés comme suit: 
a) les codes TARIC de la troisième colonne sont modifiés comme suit: 
i) à la ligne relative au code NC %quot%94051099%quot%, le code TARIC %quot%10%quot% est remplacé par le code TARIC %quot%20%quot%; 
ii) à la ligne relative au code NC %quot%94056099%quot%, le code TARIC %quot%10%quot% est remplacé par le code TARIC %quot%20%quot%; 
iii) à la ligne relative au code NC %quot%94059990%quot%, le code TARIC %quot%10%quot% est remplacé par le code TARIC %quot%20%quot%; 
b) les codes NC de la deuxième colonne sont modifiés comme suit: 
i) le code NC %quot%94051099%quot% est remplacé par le code NC %quot%94051098%quot%; 
ii) le code NC %quot%94056099%quot% est remplacé par le code NC %quot%94056080%quot%; 
iii) le code NC %quot%94059990%quot% est remplacé par le code NC %quot%94059900%quot%; 
3) Dans la deuxième partie de l'annexe IV, les codes sous le numéro d'ordre 09.0106 sont modifiés comme suit: 
a) les codes TARIC de la troisième colonne sont modifiés comme suit: 
i) à la ligne relative au code NC %quot%52085290%quot%, les codes TARIC %quot%11%quot% et %quot%91%quot% sont supprimés;

i) les codes NC %quot%52085210%quot% et %quot%52085290%quot% sont remplacés par le code NC %quot%52085200%quot%; 
ii) le code NC %quot%62044990%quot% est remplacé par le code NC %quot%62044900%quot%; 
iii) les codes NC %quot%62149010%quot% et %quot%62149090%quot% sont remplacés par le code NC %quot%62149000%quot%; 
4) Dans la deuxième partie de l'annexe V, les codes sous le numéro d'ordre 09.0103 sont modifiés comme suit: 
a) dans la troisième colonne, à la ligne relative au code NC %quot%52085290%quot%, les codes TARIC %quot%11%quot% et %quot%19%quot% sont supprimés; 
b) dans la deuxième colonne, les codes NC %quot%52085210%quot% et %quot%52085290%quot% sont remplacés par le code NC %quot%52085200%quot%.
