Bon,une chose est sûre, Dan Brown montre que ce n'est pas la peine d'être un écrivain pour pondre un bouquin qui se vend bien. Ce livre sucite des commentaires passionnés de lecteurs fanatiquement pour qui rêvent d'en découdre avec ceux des béotiens qui n'ont rien compris. PAs la peine de se battre pour ça les gars. Voilà ce qui devrait arriver normalement (si le battage médiatique - bravo le service marketing - n'avait pas été aussi important):  Comme on prend le train, on va s'acheter un bouquin (Oh tiens, le code Da Vinci, je vais le prendre pour me passer le temps) Ensuite on lit, c'est pas désagréable mais on bout d'une heure ou deux on commence à s'ennuyer un peu (le coup du code dans le code dans le code, c'est pénible à la longue). Toutefois, comme on a encore 2 heures de voyage, on le termine (ah, c'était donc ça ?) Ca n'a pas été désagréable, mais le train est arrivé, on laisse le bouquin sur le fauteuil et, à peine descendu du wagon, on l'a déja oublié.
