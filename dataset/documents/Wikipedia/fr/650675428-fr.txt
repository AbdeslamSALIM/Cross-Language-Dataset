 Ghostscript est un interpréteur PostScript libre pour toutes les plates-formes.Depuis , les nouvelles versions de Ghostscript sont distribuées exclusivement sous la licence GNU GPL ..HistoriqueHistoriquement, deux versions de Ghostscript coexistaient :AFPL Ghostscript : Développé par Artifex Software et contenant les dernières améliorations. Bien qu’il ne soit pas sous licence GNU GPL, les sources sont néanmoins disponibles.GNU Ghostscript : Développé également par Artifex Software, il est en licence GNU GPL.Interfaces graphiquesGhostscript n’est qu’un interprète.
