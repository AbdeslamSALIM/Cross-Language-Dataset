   PuTTY est un émulateur de terminal doublé d'un client pour les protocoles SSH, Telnet, rlogin, et TCP brut. Il permet également d'établir des connexions directes par liaison série RS-232. Il était à l'origine disponible uniquement pour Windows, mais il est à présent porté sur diverses plate-formes Unix (et non-officiellement sur d'autres plate-formes). PuTTY est écrit et maintenu principalement par Simon Tatham.C'est un logiciel libre distribué selon les termes de la licence MIT.CaractéristiquesAucune installation n'est nécessaire ; il suffit de lancer l'exécutable téléchargé pour l'utiliser.Supporte les protocoles Telnet, SSH et rlogin, ainsi que les liaisons RS-232.Il existe une version en ligne de commande appelée Plink.== Putty pour les téléphones mobiles ==Il existe des versions de PuTTY sur système Symbian pour des téléphones mobiles de type Nokia, et des versions pour Windows Mobile.Liens externesLogiciel d'origine Site officiel de PuTTY, avec documentation en anglais Traduction française de la documentation de PuTTYLogiciels dérivés de PuTTY PuTTY Tray, une version améliorée du client original PuTTY Connection Manager, un complément à PuTTY permettant de gérer le multi-session par onglets dans une fenêtre unique (application en français et anglais) ExtraPuTTY, est un complément à PuTTY qui permet d’automatiser des ensemble de commandes par un simple clic (création de scénarios de commandes etc).
