 XNA, cuyo significado (recursivo) es 'X'NA’s 'N'ot 'A'cronymed, es una API desarrollada por Microsoft para el desarrollo de videojuegos para las plataformas Xbox 360, Zune y WindowsTécnicamente es un Marco de Trabajo (Framework), basado en .NET Framework 2.0 y al igual el .NET Framework 2.0, éste corre sobre el CLR, aunque en una implementación que provee un manejo optimizado para la ejecución de videojuegos.Para el desarrollo sobre este Marco de Trabajo (Framework) se usa el Microsoft GameStudio el cual usa como unico lenguaje C# pero debido a la naturaleza de la filosofía de desarrollo .NET que mediante el CLR permite que todos ellos compilen a código nativo usando un mismo código intermedio (MSIL) no hay razón técnica para que no sea así. Este Marco de Trabajo (Framework) sirve para hacer conexiones con el DirectX 9.0c en tiempo de ejecución justo después de la compilación JIT.HistoriaEn el 2004 Microsoft anunció lo que sería su nuevo framework de desarrollo de videojuegos, para las plataformas Xbox 360 aquel entonces Xenon y Windows, su nombre sería XNA.Enseguida comenzó el desarrollo de lo que era conocido como el DirectX 9.0c manejado usando C#, es decir que sin ser código nativo podía usar las bibliotecas de DirectX 9.0c algo antes casi exclusivo para lenguajes como C/C++, posteriormente comenzó el desarrollo de una plataforma simplificada que redujera el largo proceso de desarrollo como resultado se obtuvo el Microsoft GameStudio Express que usaría las bibliotecas ahora llamadas XNA, en diciembre de 2007 se lanzó el MS GameStudio 2.0 que a pesar de lo que muchos decían que iba a ser dividido entre la versión express y la versión profesional se deicidio lanzar de forma conjunta eliminando diferencias y permitiendo que su instalación funcione como parche en cualquiera de las versiones de las aplicaciones de desarrollo para C# 2.0.Después de un CTP 2006 y una versión Beta par de meses después, Microsoft lanzó la primera versión de su plataforma XNA el 11 de Diciembre de 2006.Ahora han unido fuerzas con Softimage XSI Mod Tool y Valve Software para ayudar a los jugadores que quieran aprender a crear videojuegos.En Diciembre de 2007 la versión 2.0 del framework salió de su etapa beta, con una madurez mayor a la versión 1.0, y esta vez incorporando soporte para todas las versiones de Visual Studio 2005 (en la versión anterior solo Visual C# Express Edition 2005 estaba soportado).En el 30 de Octubre, se lanzó la version 3.0 del framework luego de una beta anterior, en el cual se tendria soporte para Visual Studio 2008 y tambien soporta para las nuevas caracteizticas de C# 3.0 y Linq entre otros, y con cambios en el framework, entre las nuevas caracterizticas novedosas, se incorpora la creación de juegos para Zune, el reproductor multimedia de Microsoft.Véase tambiénXbox 360 (Consola desarrollada por Microsoft).Zune (Dispositivo de multimedia desarrollado por Microsoft).Enlaces
