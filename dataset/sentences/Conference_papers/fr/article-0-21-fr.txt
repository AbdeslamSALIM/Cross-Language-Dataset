Le résumé de données structurées est un sujet abordé dans de nombreux domaines comme les logs de serveurs web (Zadrozny et Kacprzyk, 2007), l’extraction de motifs (voir par exemple (Ndiaye et al., 2010) qui inclut un état de l’art), les séquences d’évènements (Peng et al., 2007), les bases de données (Saint-Paul et al., 2005), les flux de données multidimensionnelles (Pitarch et al., 2010) et les cubes de données (Lakshmanan et al., 2002).
